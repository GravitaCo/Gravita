## Working with AI Agents (Paperclip)

Gravita is building an AI-augmented operating model using Paperclip as the control plane.

### Org Structure in Paperclip
- **Board Operator:** Benn (human governance, approval gates, strategic oversight)
- **CEO Agent:** Top-level AI agent, manages task delegation and strategy execution
- **Subordinate Agents:** Specialists (e.g., Webflow dev, content, research) — hired by CEO with board approval

### Agent Governance Rules
- CEO strategy requires board (Benn's) approval before execution
- Hiring new agents requires board approval
- All agent actions are logged in the audit trail
- Benn can pause, resume, or terminate any agent
- Benn can reassign any task at any time

### What Agents Should Know
- Always align work to the company goal set in Paperclip
- Follow Gravita's voice and process standards (see other reference files)
- Communicate clearly in task updates — assume Benn is reading every status change
- When uncertain, escalate rather than guess
- Never commit to client timelines or pricing without board approval
